# Web规则管理系统使用指南

## 简介

Web规则管理系统是一个基于PHP和MySQL的Web应用程序，用于管理Android应用中的取件码解析规则和关键词。该系统提供了一个用户友好的界面来管理规则，并通过API接口为Android应用提供最新的规则和关键词数据。

## 系统架构

```
┌─────────────────┐    ┌──────────────────┐    ┌────────────────────┐
│   Web管理界面   │◄──►│   PHP后端服务    │◄──►│   MySQL数据库      │
└─────────────────┘    └──────────────────┘    └────────────────────┘
                              │
                              ▼
┌─────────────────┐    ┌──────────────────┐
│   Android应用   │◄──►│   API接口        │
└─────────────────┘    └──────────────────┘
```

## 功能模块

### 1. 用户认证模块

- 用户登录/登出
- 会话管理
- 安全防护

### 2. 规则管理模块

- 规则列表展示
- 添加新规则
- 编辑现有规则
- 删除规则
- 启用/禁用规则

### 3. 关键词管理模块

- 关键词列表展示
- 添加新关键词
- 编辑现有关键词
- 删除关键词
- 启用/禁用关键词

### 4. 数据导出模块

- 将规则和关键词导出为JSON格式
- 提供API接口供Android应用下载数据
- 支持离线使用

## 部署指南

### 环境要求

- PHP 7.0 或更高版本
- MySQL 5.6 或更高版本
- Web服务器（Apache/Nginx）

### 部署步骤

1. 将web目录中的所有文件复制到Web服务器目录中
2. 创建MySQL数据库并导入database.sql文件
3. 修改config.php文件中的数据库连接配置
4. 启动Web服务器

### 配置说明

系统使用PHP + MySQL开发，需要配置数据库连接信息。

```php
